# Streamlit-приложение для анализа стабильности углеводородов

Интерактивный веб-интерфейс на базе Streamlit для скрипта анализа стабильности углеводородов `BS_qwen.py`. Приложение позволяет загружать данные, настраивать все параметры анализа, запускать расчеты, визуализировать результаты и сохранять конфигурации.

## Возможности

- **Загрузка данных**: Поддержка файлов Excel (.xlsx, .xls) и CSV
- **Настройка параметров**: Гибкая настройка всех параметров анализа через удобный интерфейс
- **Визуализация**: Интерактивные графики на базе Plotly
- **Сохранение конфигураций**: Возможность сохранять и загружать настройки анализа
- **Экспорт результатов**: Выгрузка результатов в Excel и скачивание графиков

## Установка

### 1. Требования

- Python 3.8 или выше
- pip (менеджер пакетов Python)

### 2. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 3. Запуск приложения

```bash
streamlit run app.py
```

После запуска приложение откроется в вашем браузере по адресу `http://localhost:8501`

## Быстрый запуск на Windows

Для удобства пользователей Windows создан скрипт `start.bat`. Просто дважды кликните по нему для запуска приложения.

Или запустите через командную строку:

```batch
start.bat
```

## Использование

### 1. Загрузка данных

1. В боковой панели нажмите "Обзор файлов" и выберите файл с данными
2. Проверьте предпросмотр данных и убедитесь, что все колонки распознаны корректно

### 2. Настройка параметров

Используйте панель параметров для настройки:
- **Основные настройки**: Выбор колонок с данными, пороги отсечения
- **Параметры методов**: Включение/выключение методов анализа
- **Оптимизация**: Настройка алгоритмов оптимизации
- **Фильтрация выбросов**: Методы и пороги фильтрации
- **Визуализация**: Настройки отображения графиков

### 3. Сохранение конфигурации

1. Введите имя конфигурации в поле в боковой панели
2. Нажмите "Сохранить конфигурацию"
3. Для загрузки ранее сохраненной конфигурации выберите её из списка и нажмите "Загрузить"

### 4. Запуск анализа

1. Нажмите кнопку "Запустить анализ"
2. Дождитесь завершения расчетов
3. Просмотрите результаты во вкладках

### 5. Экспорт результатов

- **Табличные данные**: Кнопка "Скачать результаты (Excel)"
- **Графики**: Используйте встроенную кнопку экспорта в интерфейсе каждого графика Plotly

## Формат входных данных

Приложение ожидает Excel или CSV файл со следующими данными:
- Колонки с концентрациями углеводородов
- Колонки с указанием годов/периодов измерений
- Дополнительные метаданные (опционально)

Точный формат зависит от требований скрипта `BS_qwen.py`.

## Troubleshooting

### Ошибка при запуске

Убедитесь, что все зависимости установлены:
```bash
pip install -r requirements.txt --upgrade
```

### Проблемы с кодировкой файлов

Все файлы должны быть в кодировке UTF-8. При проблемах с русскими символами проверьте кодировку входных файлов.

### Медленная работа на больших данных

Для больших датасетов время расчета может увеличиться. Используйте фильтры для ограничения объема обрабатываемых данных.
