## Запуск тестов

Для запуска тестов выполните команду:

```bash
cd /workspace
python -m pytest tests/test_app.py -v
```

Или просто:

```bash
pytest tests/test_app.py -v
```

## Рекомендации

1. **Регулярный прогон тестов**: Рекомендуется запускать тесты после каждого значительного изменения кода

3. **Интеграция в CI/CD**: Настроить автоматический запуск тестов при коммитах в репозиторий
